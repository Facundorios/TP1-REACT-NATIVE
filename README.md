# Trabajo Práctico Evaluativo: React Hooks.

## Contenido

- [CounterClick](#CounterClick): Un componente de contador simple que utiliza el hook `useState` para mantener un estado local.
- [Temporizator](#Temporizator): Un componente que muestra un temporizador en formato de horas, minutos y segundos, implementado con el hook `useState` y `useEffect`.
- [Theme](#Theme): Un componente que muestra cómo cambiar el tema de la aplicación utilizando el hook `useState` y `useContext`.

## CounterClick

El componente `CounterClick` es un ejemplo básico de cómo utilizar el hook `useState` para mantener un contador local. Al hacer clic en el botón, el contador se incrementa en uno.

## Temporizator

El componente `Temporizator` muestra un temporizador que cuenta las horas, minutos y segundos desde que se monta el componente. Utiliza el hook `useState` para mantener el estado del contador y `useEffect` para actualizar el contador cada segundo.

## Button Theme

El componente `Theme` demuestra cómo cambiar el tema de la aplicación utilizando el contexto (`useContext`). Al hacer clic en el botón, se cambia el tema entre claro y oscuro. También se muestra cómo utilizar el contexto en un componente secundario, `ThemedButton`, para cambiar su estilo según el tema seleccionado.


## Tecnologías utilizadas

- React
- JSX
- React Hooks (useState, useEffect, useContext)
- Context API
